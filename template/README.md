## Build
Run `ng build` to build the library project using ng-packagr. The build artifacts will be stored in the `dist/%PROJECT_NAME%/` directory.

## Publishing
Run `npm publish` in the `dist/%PROJECT_NAME%/` directory in order to publish the library to a package registry.
